## Deploying to Vercel

1) Create a new Vercel project and import this repo.  
2) Framework preset: **Next.js**. Build command: `npm run build`. Output: `.next`.  
3) Add environment variables if needed; none are required for the static content here.  
4) Push to the default branch and Vercel will build & deploy automatically.
